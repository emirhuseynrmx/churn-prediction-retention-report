## Not Included Unless Quoted Separately

- Live API deployment
- CRM integration
- Scheduled retraining
- Dashboard hosting
- Database or warehouse connection
- Stripe, HubSpot, Salesforce, or Intercom integration
- Monthly monitoring
- Production MLOps
- Guaranteed model accuracy
- Large-scale manual data cleaning
- SQL warehouse extraction

## Client Data Privacy

I only use client data for the project delivery. I do not resell, share, or reuse client data.

Sensitive columns can be removed or anonymized before modeling. Raw client data should not be committed to a public repository.

This package is for a one-time churn analysis and retention report from a provided CSV. API deployment, recurring automation, CRM integration, and dashboard hosting are available as custom add-ons.
